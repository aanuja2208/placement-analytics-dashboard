✨ Paytm Sorted - Technical Architecture & UX Dashboard

This interactive prototype demonstrates the front-end capabilities of Paytm Sorted. It is built to resolve the social splitting friction for the 15-25 age bracket without sacrificing Paytm's critical home screen advertising and loan cross-selling modules.

Split Score (Reputation Meter) Tracks a user's average settlement time in minutes. Slow repayment rates degrade the score dynamically. This uses peer pressure to force repayments without manual payment chasing.
NPCI UPI Request Pay API Rather than standard messages, the system triggers official UPI Collect requests using the API structure. Settlement callback automatically settles the database.
