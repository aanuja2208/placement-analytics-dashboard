## 🔧 TROUBLESHOOTING

**Missing dependencies:**
```bash
pip install -r requirements.txt --break-system-packages
```

**Data issues:**
```bash
python scripts/generate_enhanced_data.py
```

**Port in use:**
```bash
streamlit run app.py --server.port 8502
```
